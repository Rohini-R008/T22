# Tailwind Hero Landing Page

## Overview

In this project, I created a simple and clean **Hero Landing Page** using **Tailwind CSS**. The goal was to design a modern workspace landing page layout without using custom CSS, and instead rely on Tailwind utility classes.

## Features

* I implemented a **sticky navigation bar** using `sticky`, `top-0`, and `z-50`. It includes a logo, navigation links, and a “Get AI free” button.
* The **hero section** has a gradient background and centered content with a main heading and description.
* I added two **CTA buttons** (“Get AI free” and “Request a demo”) aligned horizontally.
* A **dashboard image** is placed in the center using relative positioning.
* I created three **floating cards** (Tasks, Status, Team Activity) using absolute positioning and shadow utilities.
* A **fixed chat widget** is placed at the bottom-right corner of the page.

## Technologies Used

* HTML
* Tailwind CSS (CDN)
* JavaScript (basic)

## How to Run

Open the `index.html` file in a browser. Internet is required for Tailwind CDN.

## What I learnt

Through this project, I understood how to use Tailwind CSS for layout design, positioning (relative, absolute, sticky, fixed), and creating a clean UI without writing custom CSS.
